# NotationForDummies

A very bare-bones musical editor which is more likely a test how far Gloss can be pushed.

## Install

First, you need to install cabal (Instructions are given for Fedora Workstation but cabal works on almost all distros), the compiler for haskell.

```bash
sudo dnf install cabal
cabal install gloss
```

Then, you need to clone this repo

```bash
git clone https://github.com/tanIsOnGithub/NotationForDummies
cd NotationForDummies
```
